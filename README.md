#### 关键参数

* 400 KB SRAM (TCM)，384 KB ROM
* 2.4 GHz Wi-Fi & BLE 5.0
* USB1.1
* 2 × UART
* 22  x GPIO
* -40°~85°工作温度
* QFN32 (5x5mm)
